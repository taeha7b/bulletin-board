# bulletin-board

## 프로젝트 목적
브랜디에서 인턴 기간동안 Flaks로 구현한 기능들을 Django로 구현해보기 전에 
간단하게 게시판을 제작해보면 Django의 ORM을 숙지한다.

## 적용 기술 및 구현된 기능
### 적용 기술
* Python, Django web framework
* Bcrypt
* JWT
* MySQL

### 구현된 기능
* Bcrypt를 활용한 비밀번호 암호화
* JWT를 활용한 로그인 토큰 발행
* 회원가입시 정규식을 사용한 회원정보 유효성 검사
* 게시판 작성, 수정, 논리 삭제, 읽기
